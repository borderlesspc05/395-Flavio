## Aplicação na plataforma (tema escuro)

O app mantém **dashboard escuro** (SaaS), derivando a paleta do site:

| Variável CSS | Valor | Origem |
|--------------|-------|--------|
| `--primary-dark` | `#1a222d` | Fundo profundo |
| `--primary-blue` | `#2F3A4C` | Brand primary |
| `--accent-blue` | `#AF9270` | Brand accent (ouro) |
| `--accent-vivid` | `#c9a962` | CTA hover |
| `--light-blue` | `#FFBC7D` | Realce quente |
| `--gold-logo` | `#AF9270` | Logo / detalhes |
| `--mm-brand-lavender` | `#B9B6C3` | Bordas e texto secundário |

Bordas e glows usam o ouro (`rgba(175, 146, 112, …)`) em vez do azul elétrico anterior.
